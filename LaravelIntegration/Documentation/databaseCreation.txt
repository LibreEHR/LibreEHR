This is for database creation.
------------------------------

1. Create Console command and register it.
	- php artisan make:command <command name>
	- A <command name>.php file is created in App\Console\Commands directory.
	- Write the business logic there for that particular command including signature and necessary arguments if required.
	- Register your command in Kernel.php located in App\Console directory, inside $commands.

2. Run the newly created command according to the signature you givem. In this case it is : 
	- php artisan make:database

3. Before creating this, make sure that you have properly configured your .env file.
